# Pricing Calculator — Business Logic & Decision Support

1) Objective
Develop a pricing calculator to support decision-making by applying business logic and cost-based rules, allowing users to simulate prices in a structured, clear, and consistent way.

The goal of the project is to transform inputs (costs and parameters) into reliable outputs that support pricing strategies.

2) Description
This project is a web application focused on price calculation and simulation, designed to simplify the process of defining prices based on business rules. The calculator allows users to input values, apply predefined logic, and instantly visualize results.

Although it is a front-end application, the project emphasizes data handling, calculation logic, and consistency of results, which are key aspects in data-driven decision-making.

3) Result
The final result is a functional and user-friendly pricing calculator, enabling fast simulations and standardized calculations. The application reduces manual errors and improves clarity when analyzing pricing scenarios.

4) Tools Used
React
JavaScript
HTML
CSS
5) Business Logic
The structured calculator applies rules to:

Receive cost and parameter inputs
Process calculations based on defined formulas
Return consistent and reproducible pricing results
This mirror approaches real-world scenarios where data inputs are transformed into outputs that guide business decisions.

6) Key Learnings
Implementation of business rules and calculation logic in a web application
Organization of components and state management in React
Translation of business requirements into functional logic
Improvement of usability to support decision-making
Greater understanding of how data-driven tools support operational and financial analysis
7) Possible Improvements

Add configurable parameters and scenarios
